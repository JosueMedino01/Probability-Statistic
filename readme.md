# Practice: Frequency table + charts

Small Python exercises that read `bookTable.csv` and either:

- compute a frequency table (`01.frequency/app.py`)
- generate charts (`02.charts/bar_chart.py`, `02.charts/pie_chart.py`)

## Requirements

- Python 3.x
- (optional) `make`

If the scripts use extra libraries (for example, `matplotlib` / `pandas`), install them in a virtual environment.

## Run

Using `make`:

```sh
make freq
make chart-bar
make chart-pie
```

Or run the scripts directly:

```sh
python3 01.frequency/app.py
python3 02.charts/bar_chart.py
python3 02.charts/pie_chart.py
```

## Data

- Input dataset: `bookTable.csv`
